# Описание базовых механик (Core Mechanics)

Игра представляет собой экшен с жесткой привязкой к механике эскорта и позиционирования. Геймплей строится на балансе между защитой уязвимого NPC и агрессивным ведением боя.

## 1. Система здоровья и штрафы
В игре присутствуют две независимые шкалы здоровья: **Игрок** и **эскортируемый NPC**.

* **Условие поражения:** Падение любой из двух шкал до нуля приводит к рестарту с контрольной точки.
* **Динамическая сложность (Штраф):** Если здоровье NPC падает ниже 50%, наносимый Игроком урон пропорционально снижается, а на экране генерируются визуальные помехи.
* **Раздельное лечение:** * Игрок восполняет HP за счет микро-частиц, выпадающих из убитых врагов (поощрение агрессии).
    * NPC лечится только с помощью редких расходников, добываемых в опциональных зонах (сундуках).

## 2. Система состояний (Movement & Actions)
Игрок вынужден постоянно переключаться между двумя режимами:

* **Свободный режим (Боец):** Игрок передвигается на 100% скорости, имеет доступ к атакам и рывкам (dash). NPC остается лежать в точке, где его оставили.
* **Режим переноски (Ноша):** Игрок поднимает NPC. Скорость перемещения падает на 50%, любые атаки и уклонения заблокированы. Цель фазы — безопасное маневрирование между укрытиями и препятствиями.

## 3. Боевая система и Позиционирование (Зоны)
На уровнях отсутствуют постоянные классовые навыки. Способности Игрока зависят от позиционирования на арене.

* **Нейтральная территория:** Вне специальных зон Игрок лишен способностей. Доступна только слабая базовая мили-атака с эффектом легкого отталкивания.
* **Цветные зоны (Аномалии):** Нахождение внутри определенной зоны дает доступ к уникальному набору навыков (Базовая атака, Средняя способность, Ультимейт):
    * 🔴 **Красная зона (Урон):** Атака по площади (дробовик) / Притягивание врага (лассо) / Точечный взрыв цели.
    * 🟢 **Зеленая зона (Защита):** Урон по области (лужи) / Возведение физической преграды (стена шипов) / Цепной AoE-урон (заражение).
    * 🔵 **Синяя зона (Мобильность):** Быстрые мили-удары / Рывок сквозь противников с оглушением / AoE-замедление времени.

## 4. Прогрессия и Ресурсы
* **Энергия (Мана):** Расходуется на применение Средних способностей и Ультимейтов. Добывается из поверженных противников. С переходом на новые этапы максимальный пул маны и плотность её дропа увеличиваются.
* **Ограничение перемещения:** Переход между аренами или уровнями невозможен, если Игрок не донес NPC до триггера выхода.
* **Структура:** Игра состоит из 3 Актов с постепенным масштабированием сложности (количество и типы врагов, появление экологических опасностей). В конце каждого Акта — уникальный босс-файт с разрушаемыми укрытиями.

## 5. Геймплейный цикл (Core Loop)

### Микро-цикл (Бой на арене):
1.  **Анализ:** Оценка геометрии арены, укрытий и расположения Цветных зон.
2.  **Эвакуация:** Перемещение NPC (с дебаффом скорости и без защиты) в безопасное укрытие.
3.  **Бой:** Переход в Свободный режим, перемещение между Цветными зонами для использования способностей и зачистки волн.
4.  **Сбор:** Лутинг HP-частиц и Маны, возврат к NPC для перехода на следующую арену.

### Макро-цикл (Уровень):
Зачистка обязательных арен ➔ Посещение опциональных усложненных комнат (ради лечения NPC) ➔ Доставка NPC к триггеру-гейту ➔ Босс-файт ➔ Переход в следующий Акт.
